# Privacy Policy - TRUNG_HUYEN_AI_OS

Last Updated: 2026-06-25

## Overview

TRUNG_HUYEN_AI_OS is an AI Operating System designed to support knowledge management, software development, Google Drive memory, and human-approved execution.

## Data Collection

The system only accesses data explicitly authorized by the user, including:

- Google Drive documents and folders.
- GitHub repositories and source code.
- Runtime state and system configuration.
- User instructions required for execution.

## Data Ownership

All data remains the sole property of the user.

TRUNG_HUYEN_AI_OS does not claim ownership of user content.

## Data Sharing

User data is not sold, rented, or shared with third parties.

Data is processed only to provide requested functionality.

## Human Approval

Operations that create, modify, or delete data require explicit user approval, including:

- Creating documents or folders.
- Updating GitHub repositories.
- Executing workflows.
- Modifying system configurations.

## Security

TRUNG_HUYEN_AI_OS follows these principles:

- Least privilege access.
- Human-approved execution.
- Explicit system state.
- Observability first.
- Secure credential management.

Google Drive serves as the durable memory layer, while GitHub stores source code and infrastructure definitions.
